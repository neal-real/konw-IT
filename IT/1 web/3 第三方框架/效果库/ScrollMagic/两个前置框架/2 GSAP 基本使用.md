## 使用方法

1. 引入框架
2. 使用js 方法,如下示例



## 单个元素动画

~~~js
// 1.创建TweenMax对象
/*
第一个参数: 需要执行动画的元素
第二个参数: 动画执行的时长
第三个参数: 具体执行动画的属性
* */
new TweenMax(".box", 3, {x: 500});
// 2.利用静态方法执行动画
// 从当前位置到指定位置
TweenMax.to(".box", 3, {x: 500});
// 从指定位置到当前位置
TweenMax.from(".box", 3, {x: 500});
// 从第一个指定的位置到第二个指定的位置
TweenMax.fromTo(".box", 3, {x: 500}, {x: 200});
~~~



## 多个元素动画



~~~js
/*
第一个参数: 需要执行动画的元素数组
第二个参数: 动画执行的时长
第三个参数: 具体执行动画的属性
第四个参数: 上一个元素开始后,延迟多少秒开始执行下一个元素的动画
* */
// 从当前位置到指定位置
TweenMax.staggerTo([".box1", ".box2", ".box3"], 3, {x: 500}, 3);
// 从指定位置到当前位置
TweenMax.staggerFrom([".box1", ".box2", ".box3"], 3, {x: 500}, 3);
// 从第一个指定的位置到第二个指定的位置
TweenMax.staggerFromTo([".box1", ".box2", ".box3"], 3, {x: 500}, {x: 200}, 3);
~~~
