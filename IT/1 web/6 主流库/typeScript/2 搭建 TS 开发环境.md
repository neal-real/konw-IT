1. 依赖环境  nodeJS
2. vscode 插件
   1. Prettier-code formatter 格式化
3. `npm install typescript -g`
4. 运行方法1: 
   1. `tsc 文件名.ts`  转换文件
   2. `node 文件名.js` 运行文件
5. 运行方法2: 安装工具
   1. `npm install -g ts-node`
   2. `ts-node 文件名.ts` 直接运行 ts 文件



## 其他指令

`tsc --init` 会把 ts 项目初始化一个 ts 项目, 里面会多一个 `tsconfig.ts` 文件,里面有各种关系 ts 的配置信息
