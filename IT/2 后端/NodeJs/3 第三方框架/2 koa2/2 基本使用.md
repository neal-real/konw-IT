## 如何使用Koa?

1. 手动安装手动配置
2. 利用Koa脚手架工具安装使用(Koa-generator)

## 手动安装手动配置

https://www.npmjs.com/package/koa



```js
// 1.导入Koa
const Koa = require('koa');
// 2.创建服务端实例对象
const app = new Koa();

// response
app.use(ctx => {
    ctx.body = 'Hello Koa';
});

// 3.指定监听的端口
app.listen(3000);
```



## 安装

指令:  `npm install koa`

- 默认安装 koa2  , koa1已经淘汰
