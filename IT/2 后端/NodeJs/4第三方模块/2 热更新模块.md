## 什么是nodemon？

- nodemon 是一个监视服务端应用程序文件改变的包，一旦nodemon发现我们修改了服务器文件，它就会自动重启服务. 这样我们就可以省去 ctrl+c 停止服务-> 启动服务，这个步骤



## nodemon使用

全局安全

```shell
npm i nodemon -g
# 启动服务器, 代替 node app.js
nodemon app.js
```



## nodemon持续运行工具



   ~~~js

   用法 :nodemon命令替代node命令执行文件
   ~~~
